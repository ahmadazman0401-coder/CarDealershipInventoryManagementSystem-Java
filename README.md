# Car Dealership Inventory Management System

A Java-based car dealership inventory system developed using a custom Linked List data structure. This project allows users to manage car inventory, search cars, sort inventory, calculate car loans, and save updated data into a text file.

## Features

- Add new car to inventory
- Update car price
- Remove sold car
- Search cars by brand/model
- Find cars under customer budget
- Sort inventory by manufacturing year
- Sort inventory by plate number
- Calculate monthly car loan payment
- Display inventory category summary
- Read and save car data using text file

## Technologies Used

- Java
- Custom Linked List
- File Handling
- Object-Oriented Programming (OOP)

## Data Structure Used

This project uses a custom generic Linked List to store and manage car records. The Linked List supports insertion, deletion, searching, sorting, and traversal operations.

## How to Run

1. Open the project in BlueJ, VS Code, or any Java IDE.
2. Make sure `cars.txt` is in the same folder as the Java files.
3. Run `CarDealershipMain.java`.
4. Choose an option from the menu.
